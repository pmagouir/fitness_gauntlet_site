# Fitness Gauntlet

A comprehensive fitness assessment tool that evaluates your performance across multiple domains: Strength, Power, Endurance, and Durability.

## Features

- **Multi-Domain Assessment**: Track your fitness across 4 domains with 13 different tests
- **Age & Gender Adjusted Standards**: Performance benchmarks tailored to your profile
- **Visual Analytics**: Interactive charts and graphs to visualize your progress
- **Score Calculation**: Get detailed scores (0-100) and performance levels (Basic, Athletic, Elite)
- **Progress Tracking**: Monitor completion across all test domains

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

## Usage

1. **Set Your Profile**: Enter your gender, age, and weight
2. **Enter Test Results**: Select a domain and test, then enter your result
3. **View Dashboard**: See your scores, levels, and visualizations
4. **Track Progress**: Monitor which tests you've completed across all domains

## Test Domains

- **Strength**: Deadlift, Strict Press, Pull-Ups
- **Power**: Broad Jump, Grace
- **Endurance**: 100 Burpees, 2k Row, 5k Run
- **Durability**: Farmers Carry, Dead Hang, Plank, Balance

## Performance Levels

- **Elite**: Top-tier performance
- **Athletic**: Strong performance
- **Basic**: Baseline performance
- **Below**: Below baseline
